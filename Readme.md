## Spring Boot WebSocket Appplication Demo

## Requirements

1. Java - 1.8.x

2. Maven - 3.x.x

## Steps to Setup

**1. Clone the application**

**2. Build and run the app using maven**

**3. Login as 'admin' no password**

```bash
cd ...
mvn package
java -jar target/websocket-demo-0.0.1-SNAPSHOT.jar
```
